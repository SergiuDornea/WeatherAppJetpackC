# WeatherAppJetpackC
A weather forecast app done with Jetpack Compose
Weather API by:   https://openweathermap.org

1. Splash screen with animation
2. Main screen (metric units)
3. Main screen + drop down menu (imperial units)
4. Search screen
5. Favorites screen 
6. About screen
7. Settings screen
